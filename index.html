C++ TUTORIAL - DYNAMIC CAST

@verdana · Mar 20, 2018 · 4 min read

RTTI RTTI 全称运行时类型信息（Run-time Type Identification）。 RTTI 提供了一个标准的方法用来检测运行时对象的类型。 In other words, RTTI allows programs that use pointers or references to base classes to retrieve the actual derived types of the objects to which these pointers or references refer. RTTI 通过两种运算符来实现： typeid 返回指针或引用所绑定的对象的实际类型 dynamic_cast 将基类指针或引用安全地转换为派生类的指针或引用 The dynamic_cast Operator 尝试将一个对象转换成类型更具体的对象。
